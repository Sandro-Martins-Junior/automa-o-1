# Automação de Fechamento de Planilha com Python

Projeto de automação desenvolvido em Python para simular um processo real de RPA
voltado à organização e fechamento de dados financeiros em planilhas Excel.

## Tecnologias utilizadas
- Python
- Pandas
- Excel

## Descrição do processo
A automação realiza a leitura de uma base de clientes a partir de uma planilha Excel,
processa os dados automaticamente e gera uma planilha final de fechamento,
eliminando tarefas manuais repetitivas.

## Estrutura do projeto
- appp.py → script principal da automação
- clientes_fake.xlsx → base de dados de entrada
- planilha_fechamento.xlsx → resultado gerado automaticamente

## Objetivo
Demonstrar conhecimentos práticos em automação de processos (RPA),
tratamento de dados e geração automática de relatórios,
com foco em oportunidades de RPA Júnior / Estágio.
